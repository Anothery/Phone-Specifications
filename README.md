# Compare phones app

Приложение для сравнения характеристик мобильных устройств


#### Архитектура:
MVP, Clean Architecture


#### Используемые технологии:
Kotlin

Retrofit

SQLite (Room)

RxJava 2, RxBinding

Dagger 2
